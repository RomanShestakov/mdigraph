About
=====

This is an Mnesia implementation of the core Erlang module 'digraph' from 
stdlib.

Installation
============

Prerequisites
-------------
* erlang R14A
  - kernel, stdlib, sasl, mnesia

Compilation
-----------

Build the source code by running::

  make

To clean::

  make clean
